# page.txt guide

`page.txt` controls the content of the landing page.

A project by [EmaF205](https://linktr.ee/emaf205).  
Made with ♥ in Milan.

## Main fields

```txt
SITE_TITLE:
SITE_DESCRIPTION:

EYEBROW:
HERO_TITLE:
HERO_TEXT:

CTA_TEXT:
CTA_URL:
CTA_TARGET:
```

## Repeated blocks

Use repeated blocks to create sections.

### Points

```txt
POINT: short-id
TITLE: Start from one real problem
TEXT: Bring your work, project or creative process.
```

### Steps

```txt
STEP: short-id
TITLE: Bring an idea
TEXT: Start with a problem or project you actually need.
```

### Details

```txt
DETAIL: short-id
TITLE: Format
TEXT: One-day workshop in Milan.
```

## Final CTA

```txt
FINAL_TITLE:
FINAL_TEXT:
FINAL_CTA_TEXT:
FINAL_CTA_URL:
FINAL_CTA_TARGET:
```

## Footer

```txt
FOOTER_TEXT:
```

## Optional sections

Leave a section empty or delete its blocks to hide it.

The structure stays clean without requiring layout settings.
