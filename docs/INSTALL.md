# Install

A project by [EmaF205](https://linktr.ee/emaf205).  
Made with ♥ in Milan.

## Basic install

1. Download `dist/EMAF205-Landing.zip`.
2. Unzip it on your computer.
3. Edit `page.txt`.
4. Edit `style.txt` if needed.
5. Upload the files to your server with FTP.

Example:

```txt
/public_html/my-landing/
├─ index.php
├─ page.txt
├─ style.txt
└─ custom.css
```

Open the folder URL in your browser.

## Requirements

- PHP 8 or newer recommended
- No database
- No admin panel
- No build step

## Runtime files

Required:

```txt
index.php
page.txt
style.txt
```

Optional:

```txt
custom.css
```
