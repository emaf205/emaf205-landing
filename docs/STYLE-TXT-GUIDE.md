# style.txt guide

`style.txt` controls the basic visual identity.

A project by [EmaF205](https://linktr.ee/emaf205).  
Made with ♥ in Milan.

## Fields

```txt
BACKGROUND_COLOR:
TEXT_COLOR:
MUTED_TEXT_COLOR:
FONT:

BUTTON_BACKGROUND:
BUTTON_TEXT_COLOR:
BUTTON_BORDER_COLOR:

LINE_COLOR:
```

## Font values

```txt
serif
sans
```

## Color values

Use HEX colors:

```txt
#ffffff
#111111
#666666
```

`BUTTON_BACKGROUND` can also be:

```txt
transparent
```

Invalid values fall back to safe defaults.
