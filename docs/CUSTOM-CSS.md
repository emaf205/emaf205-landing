# custom.css

`custom.css` is optional and advanced.

A project by [EmaF205](https://linktr.ee/emaf205).  
Made with ♥ in Milan.

If `custom.css` exists next to `index.php`, it is loaded automatically after the default styles.

Use it for small manual overrides.

Example:

```css
.hero {
  min-height: 86vh;
}

.points {
  grid-template-columns: 1fr;
}
```

For normal use, edit only:

```txt
page.txt
style.txt
```
